Web Exploitation


Website di seluruh dunia diprogram menggunakan berbagai bahasa pemrograman. Meskipun ada kerentanan khusus dalam setiap bahasa pemrograman yang harus diperhatikan oleh developer ketika membuat website, ada masalah mendasar pada internet yang dapat muncul terlepas dari bahasa pemrogramman atau Framework yang developer pilih ketika membuat sebuah website.

Kerentanan ini sering muncul dalam CTF sebagai tantangan keamanan web di mana pengguna perlu mengeksploitasi bug untuk mendapatkan hak istimewa tingkat yang lebih tinggi. Misalnya dari pengunjung website biasa menjadi Administrator dalam sebuah website.

Kerentanan yang umum ditemukan dalam tantangan CTF adalah:

- SQL Injection
- Command Injection
- Directory Traversal
- Cross Site Request Forgery
- Cross Site Scripting
- Server Side Request Forgery
